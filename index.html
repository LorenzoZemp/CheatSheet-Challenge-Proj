CSS CheatSheet

Selectors

Selector Example Description
* * Selects all elements
element element Selects all <element>
.class .classname Selects all elements with class="classname"
#id #idname Selects all elements with the id="idname"

Specificity

Specificity Description
.class1.class2 Selects all elements with both classname1 and classname2
.class1 .class2 Selects all elements with class="classname2" that is a descendant of classname1
<element>, .class1 With a comma, it selects both the element and class
.class1 element Selects the elements that have class1

Specificity Hierarchy

In order most most specific to least specific, the most specific will be selected first

  1. IDs (#id)
  2. Classes (.classname)
  3. Element (<element>)

Pseudo Classes

Pseudo Class Description
element:hover Activates css whenever the mouse hovers over the specified element
input:required Selects inputs that are required
input:disabled Selects every disabled specified input
input:focus Selects the input element which has focus
